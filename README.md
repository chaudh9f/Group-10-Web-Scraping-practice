# Group-10-Web-Scraping-practice
Python web scraping project using Requests, BeautifulSoup, and Pandas to extract website links and save data into CSV format.


# Web Scraping Project using Python

## Description

This project demonstrates web scraping using Python. It sends an HTTP request to a website, extracts the webpage title and hyperlinks using BeautifulSoup, and stores the collected data into a CSV file using Pandas.

The project is useful for beginners who want to learn:

- Web scraping fundamentals
- Sending HTTP requests with Requests
- Parsing HTML using BeautifulSoup
- Working with data using Pandas
- Exporting scraped data into CSV format

The script also includes basic error handling for failed requests and forbidden access responses.

---

## Technologies Used

- Python
- Requests
- BeautifulSoup4
- Pandas

---

## Features

- Sends HTTP requests to websites
- Uses custom User-Agent headers
- Extracts webpage title
- Scrapes first 20 hyperlinks from webpage
- Stores extracted data into DataFrame
- Exports data into CSV file
- Handles HTTP errors and debugging information
